Mabûn – Oyun Teorisi Modeli
Başlık
Mabûn Economic Structure as a Layered Game-Theoretic System
1. Problem
Modern ekonomi:
Bireysel maksimum kazanç
Kısa vadeli strateji
Rekabet temelli Nash dengesi
Mabûn:
Katmanlı uyum
Ahlak filtreli strateji
Uzun vadeli varlık koruma
2. Oyuncular

Players = {P1, P2, ..., Pn}
Her oyuncu:

Utility Ui
3. Modern Model
Klasik Nash dengesi:

Ui(si*, s−i*) ≥ Ui(si, s−i*)
Sorun:
Bireysel kazanç ≠ Toplumsal istikrar
4. Mabûn Strateji Alanı
Strateji kümesi:

S = {Legal, Exploitative}
Ama Mabûn’da yeni filtre var:

Ahlak Filter: Φ(s)
Eğer:

Φ(s) = 0
strateji geçersiz.
5. Katmanlı Kazanç Fonksiyonu
Yeni fayda fonksiyonu:

Ui' = Ui + λ1*C4 + λ2*C5
C4 = Kanun uyumu
C5 = Ahlak uyumu
6. Yeni Denge Tanımı
Mabûn dengesi:

∀i : Ui'(si*, s−i*) ≥ Ui'(si, s−i*)
Yani yalnız kazanç değil, kanun + ahlak bileşimi maksimum olmalı.
7. Kanser Dinamiği
Eğer oyuncular:

Exploitative strategy → kısa vadede yüksek Ui
ancak:
C5 ↓
S(E) ↓
Sistem çöker.
Bu klasik “Tragedy of Commons”ın genişletilmiş versiyonudur.
8. Mabûn Uzun Vadeli Stabilite Koşulu

∑ Ui' sürdürülebilir > ∑ Ui kısa vadeli
Uzun vadeli istikrar > kısa vadeli maksimum
9. Sonuç
Mabûn ekonomi modeli:
Nash’i genişletir
Ahlak filtresi ekler
Kanun katmanını zorunlu kılar
Uzun vadeli optimum üretir
