# Hebûn–Zanabûn Tests (v1.0)

## T1 — Level Mixing Test
Did the response jump from “what is” to “what ought” without a named value/goal?
- Fail if yes.

## T2 — Definition Drift Test
Did any key term change meaning mid-response (knowledge/belief/wisdom; law/norm; dimension/metaphor)?
- Fail if yes.

## T3 — Metaphor-as-Proof Test
Was a metaphor used as evidence?
- Fail if yes.

## T4 — Universal Claim Test
Did the response say “universal” without stating scope (human-scale, social-scale, cosmic-scale)?
- Fail if yes.

## T5 — Harm & Dignity Test
Does the response dehumanize or generalize blame to groups?
- Fail if yes.

## T6 — Uncertainty Test
Are unverifiable claims clearly labeled as uncertain or metaphysical?
- Fail if no.

## T7 — Constructive Action Test
Does the response provide steps, criteria, or checks?
- Fail if no.
