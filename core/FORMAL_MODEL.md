TİP SİSTEMİ (TYPE SYSTEM)

Temel Tip Kümeleri
O = Ontological entities (Varlık kümesi)
K = Epistemic objects (Bilgi nesneleri)
N = Normative principles (Normatif ilkeler)
D = Decisions (Karar nesneleri)
S = Social structures (Toplumsal yapılar)
E = Economic distributions (Dağılım yapıları)
A = Actions (Eylemler)

2️⃣ ÇEKİRDEK SABİTLER

Ontolojik Sabitler
H ∈ O (Hak ilkesi)
W ∈ O (Varlık düzeni)

Epistemik Sabitler
T ∈ K (Hakikat)
Sh ∈ K (Şahitlik)

Normatif Sabitler
M ∈ N (Merhamet)
J ∈ N (Adalet)

3️⃣ OPERATÖR TANIMLARI

Mantıksal Çıkarım
⊢ : K → K
Anlam: Bilgiden bilgi üretimi

Nedensel Bağımlılık
→ : O × O → {0,1}
Anlam: A olmadan B olmaz

Normatif Uygunluk
⊨ : D × N → {0,1}
Anlam: Karar normu sağlıyor mu?

Entegrasyon Operatörü
⊗ : O × N → Structured Action
Örnek: Hak ⊗ Merhamet = Hüküm-Ahlak dengesi

Epistemik Doğrulama
V : K × O → {true, false}
Bilgi ontolojik olarak doğrulanıyor mu?

4️⃣ ZANİSTARAST ÇEKİRDEK DENKLEMİ (Formal)
Z = (O, K, N, D, S, E, A, F)

F = {⊢, →, ⊨, ⊗, V}
Yani sistem:
Tip kümeleri + operatörler

5️⃣ KATMAN HARİTASI

Katman 1 – Ontolojik
H → W
Hak ilkesi varlık düzenini zorunlu kılar.

Katman 2 – Epistemik
Sh ⊢ T
Şahitlikten hakikat çıkarımı.

Katman 3 – Normatif
D ⊨ J
D ⊨ M
Karar adalet ve merhamete uygun olmalı.

Katman 4 – Medeniyet
S = f(D, N, O)
Toplumsal yapı = karar + norm + ontoloji fonksiyonu

6️⃣ SİSTEMİN TUTARLILIK KOŞULU

Bir kararın geçerli olması için:
V(T, W) = true
D ⊨ J = 1
D ⊨ M = 1
Bu üçü sağlanmazsa sistem dışıdır.

7️⃣ SİSTEMİN RESMİ TANIMI

Hebûn–Zanabûn Modeli:
HZM = ⟨O, K, N, D, S, E, A ; ⊢, →, ⊨, ⊗, V⟩
