3️⃣ OPERATÖR TANIMLARI

Mantıksal Çıkarım
⊢ : K → K
Anlam: Bilgiden bilgi üretimi

Nedensel Bağımlılık
→ : O × O → {0,1}
Anlam: A olmadan B olmaz

Normatif Uygunluk
⊨ : D × N → {0,1}
Anlam: Karar normu sağlıyor mu?

Entegrasyon Operatörü
⊗ : O × N → Structured Action
Örnek: Hak ⊗ Merhamet = Hüküm-Ahlak dengesi

Epistemik Doğrulama
V : K × O → {true, false}
Bilgi ontolojik olarak doğrulanıyor mu?
