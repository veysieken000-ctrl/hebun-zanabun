# Mabûn — Civilizational Economic Structure

## Amaç

Mabûn yalnız ekonomik model değildir.
Mabûn, varlık sahasını daraltmayan üretim düzenidir.

Ekonomi burada:

- Salt üretim değildir
- Salt tüketim değildir
- Salt para sistemi değildir

Ekonomi = varlık yönetimidir.

## 2. Katmanlı Ekonomi

1. Fizik Katman → üretim araçları
2. Biyoloji Katman → yaşam sürdürülebilirliği
3. Zihin Katman → planlama ve tasarım
4. Kanun Katman → adil dağıtım
5. Ahlak Katman → sorumluluk
6. Hüküm Katman → nihai denge

## 3. Ekonomi Meclisi

Ekonomi meclisi:

- Hebûn ontolojisini dikkate alır
- Zanabûn epistemolojisiyle karar verir
- Zanistarast üst çerçevesiyle test edilir

Amaç:
Toplumun varlık sahasını genişletmek.

## 4. Çöküş Önleme

Sistem yalnız 1–2 boyutta kalırsa:

→ İnsan biyolojik varlığa indirgenir.
→ Tüketim bağımlılığı oluşur.
→ Varlık daralır.
