# Civilizational Layer — Hak Merkezli Medeniyet Mimarisi (v1)

**Müdebbir:** Veysi yê MALA SAF

Bu katman, Zanistarast’ın “hüküm + ahlak” çekirdeğini toplum ölçeğine taşır.

## Üst Meclis Sıralaması (değişmez)
1) **Ahlak Meclisi** (üst ilke, niyet, doğruluk, fıtrat uyumu)
2) **Hüküm Meclisi** (bağlayıcı karar; hak ölçüsü)
3) **Ekonomi Meclisi (Mabûn)** (uygulama: ihtiyaç/üretim/paylaşım)

> Eğitim/sağlık vb. alt meclisler daha sonra eklenir; ekonomi modeli burada çekirdeklenir.

## Temel Kural
Hüküm ve ahlak ikili sistem gereği birlikte yürür:
- Hüküm yoksa ahlak “bağlayıcı” olmaz.
- Ahlak yoksa hüküm “hak” olmaz.
