# Zanistarast Game Theory Framework
Müdebbir: Veysi yê MALA SAF
Civilizational Model: Newroza Kawa

---

## 1. Oyuncu Tanımı

Her ajan i:

Ai = {Pi, Bi, Ci, Mi, Ri}

Pi = fiziksel çıkar
Bi = biyolojik güvenlik
Ci = bilişsel kapasite
Mi = ahlaki bütünlük
Ri = düzen uyumu

---

## 2. Strateji Kümesi

Si ∈ {Cooperate, Exploit, Neutral}

Cooperate = ahlaki uyum
Exploit = kısa vadeli çıkar
Neutral = sistem dışı kalma

---

## 3. Fayda Fonksiyonu

Ui = α1*Pi + α2*Bi + α3*Ci + α4*Mi + α5*Ri

Zanistarast varsayımı:

Uzun vadede:

If Mi → 0
Then Ui(t→∞) → 0

---

## 4. Nash Dengesi Analizi

Klasik oyun teorisinde:
Exploit kısa vadede kazançlı olabilir.

Zanistarast genişletmesi:

Long-term Stability Condition:

Stability = f(mean(Mi), RegulatoryIntegrity)

If mean(Mi) < Threshold
System Collapse Probability ↑

---

## 5. Tekrar Eden Oyun Modeli

Repeated Game:

t = 1 → ∞

Exploit strategy:
Short-term gain
Long-term collapse risk

Cooperate strategy:
Moderate gain
High systemic survival

---

## 6. Çöküş Fonksiyonu

CollapseRisk = β1*(1 - mean(Mi)) + β2*(TrustDecay)

If CollapseRisk > CriticalPoint
System → Structural Breakdown

---

## 7. Falsifiye Edilebilirlik

Model yanlışlanır eğer:

- Ahlaki düşük toplumlar uzun vadede daha stabil çıkarsa
- Exploit stratejisi sistem çöküşüne yol açmazsa
- Güven kaybı sistem dayanıklılığını etkilemezse

---

## Sonuç

Zanistarast oyun teorisi:

- Ahlakı stratejik merkez yapar
- Uzun vadeli istikrarı ölçer
- Sistem çöküşünü matematiksel modele bağlar

Bu model simülasyona uygundur.
